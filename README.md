## How it works

1. Bot listens for messages in the `Internal Network Ads` Telegram group (`TARGET_CHAT_ID`)
2. Each message is tested against the standard ad format:
   ```
   {Client} - {Category} - ${Price}
   @admin
   **INSTRUCTIONS:**
   - feed / reels / carousel
   - 30min NIF / Perm post
   **PAGE INFO:**
   [time] AZ
   @{page_handle} - ${price}
   ```
3. If it matches, it extracts: Client, Category, Price, Page Handle, Post Type, Post Duration, Date
4. Appends a row to the Master Revenue Sheet and (if the handle is in `config/pages.json`) the individual page sheet
